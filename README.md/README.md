# Insurance Analysis

## Hur man kör projektet
1. Öppna filen `Insurance-analys-R.Rproj` i RStudio  
2. Öppna `analysis.Rmd`  
3. Klicka på **Knit** för att köra hela analysen  

## Paket som används
- tidyverse  

## Projektstruktur
- data/ – dataset (insurance_costs.csv)  
- kod/ – R-script för analys  
- resultat/ – sparade figurer  
- rapport.docx – sammanfattning av analys
