# Insurance Analysis

## Syfte
Syftet med projektet är att analysera vilka faktorer som påverkar försäkringskostnader och bygga en regressionsmodell i R.

## Hur man kör projektet
1. Öppna `Insurance-analys-R.Rproj` i RStudio  
2. Kör skripten i mappen `kod/` i ordning:
   - 01_las_in_data.R
   - 02_datastadning.R
   - 03_beskrivande_analys.R
   - 04_regression.R  

## Paket som används
- tidyverse  

## Projektstruktur
- data/ – dataset  
- kod/ – R-script  
- resultat/ – figurer  
- rapport.docx – slutrapport
